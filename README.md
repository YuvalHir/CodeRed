# Code Red - Productivity Timer for Students

A powerful Pomodoro-style productivity timer designed specifically for university students facing exam periods. Built with military-inspired DEFCON levels to motivate intense focus sessions and maintain productivity during critical study times.

## üéØ Mission Statement

Code Red was created during a university exam period to help students maintain focus and productivity. The app combines time management with distraction tracking, music integration, and gamification elements to make studying more engaging and effective.

## ‚ú® Features

### Core Functionality
- **Pomodoro Timer**: Customizable focus and break times with anti-quit protection
- **DEFCON Alert System**: Visual escalation from calm (DEFCON 5) to critical (DEFCON 1) based on streak performance
- **Task Management**: Mission log for organizing study objectives with completion tracking
- **Distraction Tracking**: Mental waste bunker to offload distracting thoughts
- **Streak Counter**: Gamified progress tracking with visual fire effects

### Advanced Features
- **Spotify Integration**: Choose from curated study playlists or add custom Spotify embeds
- **Real-time Countdown**: Set target dates and track time remaining
- **Hebrew RTL Support**: Full right-to-left layout for Hebrew users
- **Persistent State**: All settings and data saved across browser sessions
- **Fullscreen Mode**: Immersive study environment
- **Responsive Design**: Works on desktop and mobile devices

### Why Use Code Red?
- **Gamified Focus**: DEFCON system keeps you engaged during long study sessions
- **Distraction Management**: Built-in system to acknowledge and set aside distractions
- **Music Integration**: Curated study playlists to enhance concentration
- **Progress Tracking**: Visual feedback on study streaks and task completion
- **University Optimized**: Designed specifically for exam period productivity

## üéÆ How to Use

1. **Set Your Focus Time**: Use the Settings modal to customize Pomodoro durations
2. **Plan Your Missions**: Add study objectives to the Mission Log
